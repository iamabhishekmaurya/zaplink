# Design System Tokens

## Colors
- **Primary**: `var(--primary)`
- **Background**: `var(--background)`
- **Card**: `var(--card)`
- **Palette Defaults**: `src/features/bio/lib/templates.ts`

## Templates
Currently implemented templates:
1. **Minimal**: Clean, white/black, simple borders.
2. **Commerce**: Blue accents, optimized for product cards.
3. **Portfolio**: Dark mode, sharp edges, image backgrounds.
4. **Gradient**: Colorful gradient background, glassmorphism.
5. **Compact**: Density optimized, small rows.
6. **Editorial**: Serif fonts, rose/red palette.
7. **Neon**: Dark, high contrast green accents.

## Components
Shared components located in `@/components/ui`:
- `Button`
- `Card`
- `Dialog`
- `Input`
- `Switch`
- `Tabs`

## Icons
Using `lucide-react` for consistent iconography.
